Microprocessor vs microcontroller: just cpu is microprocessor and cpu with other periferal devices like i/o port,RAM, ROM, timer, counter, Interrupt controller
is microcontroller or microcomputer.
Counter vs timer: timer counts clockcycle whereas counter keep track of events.
word size: The processing capacity of ALU.(8-bit for 8085)
1 byte has two nibbles: lower and heigher
Machines code: Binary code that the processor understands.
Assembly code has Mnemonics. These are 3-4 letters english code. Assembly code has direct relationship with machine code. The biggest advantage of this is 
it is processor specific.
High level languages: It is converted to machine code using interpretor(python) or compiler(C/C++).
